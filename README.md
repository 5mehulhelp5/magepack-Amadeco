Magepack uses a configuration file `magepack.config.js` in your project root. While this file is automatically generated by the `magepack generate` command, you can also create or modify it manually to control advanced settings like module exclusions.

### Excluding Modules
Sometimes specific modules (e.g., payment gateways, buggy 3rd-party extensions) break when bundled or need to remain separate. You can exclude them by adding an `exclusions` array to `magepack.config.js`.

**1. Create or edit `magepack.config.js` manually:**

```javascript
module.exports = {
    // Modules starting with these strings will be ignored by the collector
    exclusions: [
        'Magento_Paypal/',
        'Amazon_Payment/',
        'Vertex_',
        'Temando_Shipping',
        'Some_Buggy_Extension'
    ],
    // The bundles array is managed automatically by the generator
    bundles: [] 
};
